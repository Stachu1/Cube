# Cube
A simple [Zig](https://ziglang.org/) program that draws a spinning cube to the terminal

<br>
<br>

## Download
```bash
git clone https://github.com/Stachu1/Cube.git
```
<br>
<br>

## Run
```bash
cd Cube
```
```bash
zig run ./src/main.zig 
```
